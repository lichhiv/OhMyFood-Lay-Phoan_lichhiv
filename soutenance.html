Présentation OhMyFood

Projet de commande de repas en ligne pour restaurants gastronomiques

1. Introduction

Contexte du projet

  • Startup de commande de repas gastronomiques en ligne
  • Expansion de New York vers Paris
  • Réduction du temps d'attente dans les restaurants

Positionnement

  • Marché de niche : restaurants gastronomiques
  • Services haut de gamme
  • Cible : classes moyennes et supérieures, connectées

2. Spécifications Techniques

HTML5
CSS3 / SASS
Mobile First
Git

Contraintes techniques

  • Développement sans JavaScript
  • Pas de frameworks CSS
  • Compatible Chrome et Firefox
  • Validation W3C HTML/CSS

3. Architecture du Projet

Structure du site

  • Page d'accueil
  • 4 pages de menus de restaurants
  • Header et Footer communs
  • Navigation intuitive

4. Fonctionnalités Clés

Loader

  • Animation de 2 secondes
  • Design aux couleurs de la charte graphique
  • Transition fluide

Animations

  • Effets hover sur les boutons
  • Animation du cœur "J'aime"
  • Transitions sur les menus

Responsive Design

  • Approche Mobile First
  • Adaptation tablette et desktop
  • Breakpoints optimisés

5. Défis et Solutions

Défis rencontrés

  • Animations CSS sans JavaScript
  • Performance du loader
  • Responsive design complexe

Solutions apportées

  • Utilisation de transitions CSS
  • Optimisation des animations
  • Media queries adaptatives

6. Perspectives d'Évolution

  • Système de réservation
  • Composition de menus personnalisés
  • Géolocalisation des restaurants

7. Conclusion

Points forts du projet

  • Interface utilisateur intuitive
  • Animations fluides
  • Design responsive
  • Performance optimisée
